## 🔄 移行・更新方法

### unseiusagi.xyz をクリスマス版に更新する場合:

```
1. GitHubで既存リポジトリにアクセス

2. 一時的にブランチ作成:
   git checkout -b christmas-special

3. 新ファイルで置き換え:
   - public/index.html → クリスマス版
   - server.js → クリスマス版

4. Vercelで確認

5. 問題なければmainにマージ

6. 1月に元に戻す
```

### 簡単な切り替え方法:

```html
<!-- index.htmlの最初に追加 -->
<script>
  const today = new Date();
  const month = today.getMonth() + 1; // 1-12
  
  if (month === 12) {
    // 12月ならクリスマス版にリダイレクト
    window.location.href = 'https://christmas-love.vercel.app';
  }
</script>
```
